Como obter o Token:
1. Acesse developers.facebook.com → Ferramentas → Explorador de API do Graph
2. Selecione seu App e gere token com permissões: ads_read e ads_management

4. O Ad Account ID está em Meta Ads Manager → URL da conta (formato act_XXXXXXXXX)

🔍 Google Ads — Planilha de Dados
Não conectada
Como conectar (sem API, grátis): 1. Abra uma planilha nova no Google Sheets
2. Clique em Extensões → Complementos → Instalar complementos e pesquise "Google Ads"
3. Instale o complemento oficial e clique em Extensões → Google Ads → Criar relatório
4. Configure com colunas: Campanha, Status, Tipo, Impressões, Cliques, CTR, CPC médio, Custo, Conversões, Taxa de conv., Custo/conv., Valor conv. total, ROAS
5. Execute o relatório, depois Compartilhar → Qualquer pessoa com o link → Leitor
6. Cole o ID da planilha abaixo e clique Salvar
💡 O ID está na URL: docs.google.com/spreadsheets/d/SEU_ID/edit
